# Explicit Review Default

## Goal

When the user explicitly invokes `audit-prompt-caching` and asks only for a review, the skill should default to a cache-focused review rather than a generic code review.

## Scope

- Add tests for the explicit review behavior.
- Add a rule to `SKILL.md`.
- Add behavioral and trigger eval coverage.

## Out Of Scope

- No script behavior changes.
- No provider reference updates.
- No changes to generic Codex review behavior outside this skill.
