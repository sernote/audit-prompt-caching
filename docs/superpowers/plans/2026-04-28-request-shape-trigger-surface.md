# Request-Shape Trigger Surface

## Goal

Make `audit-prompt-caching` activate for the changes that most often break LLM cache reuse before users explicitly mention prompt caching.

## Scope

- Expand `SKILL.md` trigger language from explicit cache symptoms to LLM request-shape changes.
- Keep the boundary clear: ordinary prompt writing, short prompt edits, and generic agent performance should not trigger.
- Add trigger eval cases for prompt text changes, message ordering, request builders, tool/schema serialization, provider surface changes, routing/deployment changes, and LLM cost regressions on repeated long requests.
- Add tests that lock in this trigger vocabulary.

## Out Of Scope

- No script behavior changes.
- No provider reference updates.
- No rename, release, or distribution work.
