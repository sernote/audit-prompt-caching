# Agent-First Skill Quality

## Goal

Make `audit-prompt-caching` stronger as an agent-facing skill: faster to trigger, clearer to apply, and harder for agents to answer with vague prompt-cache advice.

## Scope

- Add deterministic tests for the agent-first contracts and eval coverage.
- Add explicit output contracts to `audit-prompt-caching/SKILL.md`.
- Add common audit playbooks for real support/review scenarios.
- Extend the report template so handoff artifacts include agent-loop and "not worth caching" outcomes.
- Add eval prompts that exercise yes/no project-change decisions and OpenAI-compatible wrapper ambiguity.

## Out Of Scope

- No `docs/superpowers/specs/` spec document.
- No demo/GIF/value-showcase work.
- No GitHub distribution, topics, social copy, or community templates.
- No script behavior changes unless tests expose a local packaging issue.
